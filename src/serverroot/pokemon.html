Браузерная игра Покемоны.

Браузерная игра, собранная мной на фреймворке Vue.js. Использует HTML, CSS, JS и открытое API Покемонов (база данных и картинки).

Развернута на сервере GitHub Pages.

Содержание игры.

Можно искать, ловить и прокачивать покемонов. В процессе игры повышается уровень игрока. По достижении максимального уровня, игра завершается - выводится окно результатов. Доп. возможности: - Авто-сохранение игры в localStorage браузера. - Звуковая дорожка, прослушивание и выгрузка. - Выгрузка своих покемонов в PDF. - Работает в браузерах Chrome, Firefox, Edge.

Вход: Покемоны.
