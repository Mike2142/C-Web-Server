Сервер Jenkins.

Сервер автоматизации Jenkins.

Развернут на домашней системе MacOS, из контейнеров docker-compose в рамках виртуальной системы Linux CentOS.

На сервере Jenkins работает следующая задача (job): Пайплайн CI/CD для приложения Java Maven. - Скачивает исходный код Java-приложения из репозитория Github. - Запускает конвейер согласно файлу Jenkinsfile. - Стадия BUILD: Сборка приложения в jar файл, без тестов (mvn -B -DskipTests clean package). - Стадия TEST: Проводит тесты JUnit (mvn test), сохраняет результаты в формате XML и выводит их в кабинете Jenkins. - Стадия DELIVER: Запускает скрипт доставки (deliver.sh). Скрипт сохраняет приложение (jar) в локальный репо для Maven-пакетов и запускает приложение (java jar).
