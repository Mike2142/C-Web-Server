Сервер Jenkins.

Сервер автоматизации Jenkins.

Развернут на домашней системе MacOS, из контейнеров docker-compose в рамках виртуальной системы Linux CentOS.

На сервере Jenkins работают следующие задачи (jobs): - Демо-задача. Создает текстовый файл hello-world.txt и выводит его в консоль. Запускается каждые 2 минуты. - Pipeline Java Maven. Конвейер сборки приложения Java Maven. Запускается вручную. - Скачивает исходный код Java-приложения из репозитория Github. - Запускает конвейер согласно файлу Jenkinsfile. - Собирает приложение в jar файл.
