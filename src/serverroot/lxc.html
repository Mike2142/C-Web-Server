Проект LXC (Linux Containers).

Контейнер Linux Ubuntu, внутри виртуального хоста Linux Ubuntu.

Проект включает: - Хост Ubuntu 22.04. - Контейнер LXC Ubuntu Jammy amd64. - Автозапуск контейнера. - Доступность в локальной сети. - Подключение постоянного раздела на диске хоста (volume).
