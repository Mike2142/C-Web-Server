Инфраструктура Terraform.

Сеть виртуальных машин Linux, созданная в облаке Yandex Cloud с помощью инструмента Terraform.

Облачная инфраструктура включает: - Сеть, подсеть, шлюз, DNS-сервер. - VM Ubuntu1. - Параметры: CPU 2, Произв. 5%, RAM 2, Disk 8. - Подключение: ssh guest@158.160.139.253 - Пароль: guest - VM Ubuntu2. - Параметры: CPU 2, Произв. 5%, RAM 2, Disk 8. - Подключение: ssh guest@158.160.190.244 - Пароль: guest
